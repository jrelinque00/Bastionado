Dentro de Windows tenemos varias opciones de inicio de sesión para nuestro usuarios, desde la más típica, la contraseña, hasta el pin o por reconocimiento facial. Cada una de estas opciones deben elegirse según nos convenga dependiendo de cuanta seguridad necesitamos para nuesto usuario. Para poder cambiar el tipode inicio de sesión de nuestro usuario es muy simple, nos vamos a Configuración-Cuentas-Opciones de inicio de sesión y ya aquí elegimos la cual nos parezca mejor.



Una vez que tengamos nuesstro usuario de Windows con una contraseña debemos intentar securizarlo haciendo imposible el cambio de contraseña, es decir que no se pueda cambiar la contraseña del usuario. Para ello abrimos una terminal(CMD) y colocamos el siguiente comando: net user usuario /PasswordChg:No
