BIOS es la abreviatura de Binary Input Output System, y es un software que reside en un chip instalado en la motherboard de la PC, y que realiza su tarea apenas presionamos el botón de encendido del equipo.



¿Cómo funciona la BIOS?

El propósito de esta pieza de hardware es, una vez que prendemos la computadora, inicializar y configurar, además de testear si encuentre en buen estado de funcionamiento, todo el hardware del sistema, incluyendo la memoria RAM, discos duros y demás placas del sistema, para una vez finalizado este paso, cargar el gestor de arranque para que comience a ejecutarse el sistema operativo predeterminado de la computadora.

¿Cómo entrar en la BIOS?

Nota - Este proceso que se va a explicar esta reproducido con el portátil ThinkCentre M920t de Lenovo, cada ordenador tiene una manera diferente de entrar y una BIOS diferente.

Para entrar en la BIOS hay que realizar los siguientes pasos:

  1. Apagar el ordenador, si ya esta apagado seguir al paso 2.
  2. Encender el ordenador.
  3. Nada más darle al botón encender del ordenador mantener pulsado la tecla Fn y pulsar repetidamente la tecla F2.
  4. Si has seguido los pasos anteriores correctamente debería parecerte la BIOS.

Para poder controlar quién puede arrancar el sistema deberemos poner una contraseña de arranque para así evitar arranque de sistemas no deseados. Una vez accedido a la BIOS del ordenador debes seguir estos pasos:

  1. Dentro de la BIOS accedemos a la opción Security moviéndonos con las teclas de direcciones.



  2. Una vez en el panel de security nos dirigimos a la opción Set Power-On Password con la flecha de dirección abajo.



  3. Cuando estemos encima de Set Power-On Password pulsamos Enter y nos pedirá que pongamos una contraseña y la confirmemos,una vez puesta la contraseña damos a YES.



  4. Por último, Nos dirijimos a la ventana Exit y en la primera opción Save Changes and Exit le damos a Enter para salir de la BIOS y guardar los datos.

Para poder controlar quién entra en la BIOS deberemos poner una contraseña para así evitar accesos no deseados. Una vez accedido a la BIOS del ordenador debes seguir estos pasos:

  1. Dentro de la BIOS accedemos a la opción Security moviéndonos con las teclas de direcciones.



  2. Una vez en el panel de security nos dirigimos a la opción Set Administrator Password con la flecha de dirección abajo.



  3. Cuando estemos encima de Set Administrator Password pulsamos Enter y nos pedirá que pongamos una contraseña y la confirmemos,una vez puesta la contraseña damos a YES.



  4. Por último, Nos dirijimos a la ventana Exit y en la primera opción Save Changes and Exit le damos a Enter para salir de la BIOS y guardar los datos.

A continuación aprenderemos a controlar que nuestro ordenador no sea iniciado desde un sistema booteable. Una vez accedido a la BIOS del ordenador debes seguir estos pasos:

  1. Dentro de la BIOS accedemos a la opción Security moviéndonos con las teclas de direcciones.



  2. Una vez en el panel de security nos dirigimos a la opción Secure Boot con la flecha de dirección abajo y pulsamos Enter y entraremos en otra ventana.



  3. Cuando estemos dentro nos aparecerá una ventana pequeña con dos opciones Disable y Enable, seleccionamos la opción Enable y le damos a Enter.



  4. Por último, Nos dirijimos a la ventana Exit y en la primera opción Save Changes and Exit le damos a Enter para salir de la BIOS y guardar los datos.

A continuación aprenderemos a establecer un orden de arranque seguro. Una vez accedido a la BIOS del ordenador debes seguir estos pasos:

  1. Dentro de la BIOS accedemos a la opción Startup moviéndonos con las teclas de direcciones.



  2. Cuando estemos en la ventana de Starup seleccionamos Primary Boot Sequence y le damos a Enter, nos aparacerá la siguiente ventana.



  3. Cuando estemos dentro de la ventana lo único que habría que hacer es ir seleccionando cada una de las entradas y excluirlas, seleccionándolas y tecleando la tecla X, todas execpto la entrada en la que tengamos nuestro sistema operativo.

A continuación aprenderemos a bloquear el arranque F12 Boot con la contraseña de administrador de la BIOS, para ello necesitamos tener primero configurado la contraseña de administrador (paso 3). Una vez accedido a la BIOS del ordenador debes seguir estos pasos:

  1. Dentro de la BIOS accedemos a la opción Security moviéndonos con las teclas de direcciones.



  2. Una vez en el panel de security nos dirigimos a la opción RequireAdmin. Pass. for F12 Boot con la flecha de dirección abajo y pulsamos Enter y nos encontraremos en otra ventana.



  3. Cuando estemos dentro nos aparecerá una ventana pequeña con dos opciones No y Yes, seleccionamos la opción Yes y le damos a Enter.



  4. Por último, Nos dirijimos a la ventana Exit y en la primera opción Save Changes and Exit le damos a Enter para salir de la BIOS y guardar los datos.
